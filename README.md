# Conta Bancária em Java

Projeto em Java desenvolvido para praticar lógica de programação e orientação a objetos.

## Funcionalidades
- Consultar saldo
- Consultar cheque especial
- Depositar dinheiro
- Sacar dinheiro
- Pagar boleto
- Verificar uso do cheque especial

## Tecnologias
- Java
- IntelliJ IDEA
- Git e GitHub
